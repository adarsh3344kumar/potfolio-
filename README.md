# My Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Features

- **Responsive design** – looks great on all screen sizes
- **Smooth scroll navigation** with fixed navbar
- **Hero section** with gradient name effect
- **About** section with avatar
- **Skills** section with hover-animated cards
- **Projects** section showcasing work
- **Contact form** with client-side validation
- **Social links** (GitHub, LinkedIn, Email)
- **Scroll-reveal animations** via IntersectionObserver

## Project Structure

```
portfolio/
├── index.html   # Main HTML file
├── style.css    # Styles (dark theme, responsive)
└── script.js    # Interactivity (nav, form, animations)
```

## Usage

Open `index.html` in any modern browser — no build step required.
